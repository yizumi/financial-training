# 5. 損益計算書 (P/L) の分析 | Analyzing the Income Statement

<div align="right">Yusuke Izumi, CFA 著</div>

> **コードがプロダクトの設計図なら、損益計算書はビジネスの設計図だ。**

## はじめに | Introduction

ソフトウェアエンジニアや技術リーダーである私たちは、プロダクトのアーキテクチャやコードの品質に深く関心を持っています。しかし、ビジネスの持続可能性を理解するためには、もう一つの重要な「設計図」を読み解く能力が不可欠です。それが **損益計算書（Profit and Loss Statement, P/L）** です。

損益計算書は、一定期間（例えば 1 年や四半期）における会社の経営成績、つまり **「どれだけ稼いで、どれだけ費用を使い、最終的にいくら儲かったのか」** を示す財務諸表です。

- 新規機能の開発投資は、P/L のどこに影響するのか？
- マーケティングキャンペーンの費用対効果はどうだったのか？
- サーバー費用は、利益をどれくらい圧迫しているのか？

これらの問いに答えるヒントは、すべて損益計算書の中にあります。この章では、日本の有名テック企業である **株式会社メルカリ** の実際の損益計算書を例に、ビジネスのストーリーを読み解く方法を学びます。

## 5.1. 損益計算書の構造：メルカリの例

まず、メルカリの損益計算書（連結）を簡略化したものを見てみましょう。ここから、ビジネスの全体像を掴みます。

**株式会社メルカリ 連結損益計算書（簡略版）**
_(2023年6月期 通期)_

| 項目                                            | 金額（百万円） | 説明                               |
| :---------------------------------------------- | -------------: | :--------------------------------- |
| **売上高 (Revenue)**                            |    **172,060** | **事業全体の収益**                 |
| 売上原価 (Cost of Revenue)                      |         41,757 | 売上を上げるために直接かかった費用 |
| **売上総利益 (Gross Profit)**                   |    **130,303** | **本業の粗利益。粗利率 75.7%**     |
|                                                 |                |                                    |
| **販売費及び一般管理費 (SG&A)**                 |    **125,758** | **事業を運営するための費用**       |
| ├ 広告宣伝費                                    |         27,294 | マーケティング費用                 |
| ├ 支払手数料                                    |         27,381 | 決済手数料など                     |
| ├ 人件費                                        |         35,992 | 従業員の給与など                   |
| └ その他                                        |         35,091 | R&D、オフィス賃料など              |
|                                                 |                |                                    |
| **営業利益 (Operating Income)**                 |      **4,545** | **本業での儲け**                   |
| 営業外損益                                      |        △ 2,829 | 投資の評価損など                   |
| **経常利益 (Ordinary Income)**                  |      **1,716** | **本業＋財務活動での儲け**         |
| 特別損益                                        |        △ 3,115 | 一時的な損失                       |
| 税金                                            |          5,618 | 法人税など                         |
| **親会社株主に帰属する当期純利益 (Net Income)** |    **△ 7,017** | **最終的な赤字**                   |

_（注）実際の数値から教育目的で簡略化・項目整理しています。_

この表から、メルカリが **高い粗利率（75.7%）** を誇る一方で、**大規模な販管費** を投下しており、最終的に赤字となっている構造が見て取れます。

## 5.2. 売上高：ビジネスモデルの鏡

売上高は、企業がどのようにして収益を得ているかを示します。これはビジネスモデルそのものです。

> **📖 Example: テック企業の多様なビジネスモデル**
>
> - **メルカリ（マーケットプレイス型）**
>   - **収益源**: 商品が売れたときの **販売手数料** (例: 10%)
>   - 特徴：取引高（GMV: Gross Merchandise Value）が重要指標。ユーザー数と取引の活性化が売上を左右する。
> - **freee 株式会社（SaaS 型）**
>   - **収益源**: クラウド会計ソフトの **月額・年額利用料（サブスクリプション）**
>   - 特徴：継続的な収益（ARR: Annual Recurring Revenue）が安定性の鍵。顧客維持率（Retention Rate）が重要。
> - **株式会社ディー・エヌ・エー（DeNA）（ゲーム事業）**
>   - **収益源**: ゲーム内での **アイテム課金**
>   - 特徴：ヒット作に依存しやすいが、成功すれば高い利益率。DAU/MAU や課金率（ARPPU）が重要指標。

このように、同じ「売上高」という項目でも、その源泉はビジネスモデルによって全く異なります。自社のビジネスモデルがどのタイプに属し、主要な KPI が何かを理解することが第一歩です。

## 5.3. 費用：事業活動のコスト

費用は、売上を生み出すために企業がどのような活動に投資しているかを示します。特にテック企業では、費用の内訳が重要です。

### 売上原価 (Cost of Revenue)

売上原価は、売上に直接連動するコストです。

- **メルカリの場合**: 決済代行会社に支払う **決済手数料**、ユーザーが CS に問い合わせる際の **カスタマーサポート費用**、サービスを安定稼働させるための **サーバー費用（クラウドインフラ費）** などが含まれます。
- **製造業との違い**: 物理的な製品を作るメーカーの「原材料費」とは異なり、プラットフォームビジネスではサービス提供のためのインフラ費用などが原価となります。

### 販売費及び一般管理費 (SG&A)

販管費は、事業全体を運営するための費用です。テック企業では、ここが最も重要で、戦略的な投資が反映されます。

- **広告宣伝費 (Advertising)**
  - メルカリが **約 273 億円** を投下しているのは、新規ユーザー獲得とブランド維持のためです。TVCM や Web 広告などがこれにあたります。エンジニアとしては、「この広告費で獲得したユーザーの LTV (顧客生涯価値) は見合っているのか？」という視点を持つと面白いでしょう。
- **人件費 (Personnel / Salary)**
  - エンジニア、デザイナー、プロダクトマネージャー、営業、管理部門など、全従業員の給与や賞与です。メルカリでは **約 360 億円** に上り、最大の費用項目の一つです。優秀な人材の獲得と維持がいかに重要かがわかります。
- **研究開発費 (R&D)**
  - 将来のプロダクトやサービスのための先行投資です。例えば、メルカリの AI 出品機能や、メルコインのような新規事業の開発費用がここに含まれます。P/L 上は費用として計上されますが、未来の売上を生むための **戦略的投資** と言えます。

## 5.4. 利益：企業の成績表

利益にはいくつかの段階があり、それぞれが異なる意味を持ちます。

1.  **売上総利益 (Gross Profit)**
    - `売上高 - 売上原価`
    - **本業の基本的な収益力** を示します。メルカリの粗利率は 75.7% と非常に高く、これは手数料ビジネスの利益率の高さを物語っています。
2.  **営業利益 (Operating Income)**
    - `売上総利益 - 販管費`
    - **本業で稼いだ最終的な利益** です。メルカリはこの段階で **45 億円** の黒字を確保しており、本業のビジネスが健全に回っていることを示しています。
3.  **経常利益 (Ordinary Income)**
    - `営業利益 + 営業外損益`
    - 本業以外の財務活動（例：銀行からの借入利息、保有株式の配当金）を含めた利益です。
4.  **当期純利益 (Net Income)**
    - `経常利益 + 特別損益 - 税金`
    - **会社に残る最終的な利益（または損失）** です。メルカリは最終的に **70 億円の赤字** でしたが、これは一時的な損失（特別損失）や税金の支払いが影響しています。営業利益が黒字であるため、過度に悲観する必要はないかもしれません。

技術リーダーとしては、特に **営業利益** に注目すべきです。自分たちの開発したプロダクトや機能が、最終的に本業の利益にどう貢献しているかを意識することが、ビジネス視点を持つ上で重要です。

## まとめ | Summary

**重要なポイント：**

1.  **損益計算書 (P/L) はビジネスの設計図**：一定期間の経営成績（収益、費用、利益）を示す。
2.  **売上高はビジネスモデルを映す鏡**：手数料、サブスク、課金など、収益源は企業によって多様。
3.  **費用には戦略が表れる**：テック企業では、人件費、広告宣伝費、研究開発費が重要な戦略的投資。
4.  **利益は多段階で見る**：売上総利益（本業の収益力）、営業利益（本業の最終利益）、当期純利益（最終的な儲け）のそれぞれに意味がある。
5.  **営業利益を重視する**：技術リーダーとして、自分たちの活動が本業の利益にどう貢献しているかを把握することが重要。

## 演習問題 | Practice Problems

---

### 問題 1：利益の計算

ある SaaS 企業の P/L が以下の通りです。この企業の営業利益はいくらでしょうか？

- 売上高：5,000 万円
- 売上原価：1,000 万円
- 販管費：2,500 万円
- 営業外収益：200 万円

- A. 1,500 万円
- B. 1,700 万円
- C. 2,500 万円

---

### 問題 2：ビジネスモデルの推測

以下の P/L を持つ A 社と B 社は、それぞれどのようなビジネスモデルの可能性が高いでしょうか？

**A 社**

- 売上高: 100 億円
- 売上原価: 80 億円
- 売上総利益: 20 億円 (粗利率 20%)

**B 社**

- 売上高: 100 億円
- 売上原価: 15 億円
- 売上総利益: 85 億円 (粗利率 85%)

- A. A 社: 小売業、B 社: SaaS 企業
- B. A 社: SaaS 企業、B 社: 小売業
- C. 両社とも同じビジネスモデルである

---

### 問題 3：戦略的投資の分析

あるゲーム会社が、今期の広告宣伝費を昨年の 10 億円から 30 億円に増やしました。その結果、売上は 50 億円から 70 億円に増加しましたが、営業利益は 20 億円から 15 億円に減少しました。この状況から読み取れることとして、最も適切なものはどれですか？

- A. 広告宣伝費の増加は完全に失敗だった。
- B. 短期的な利益は犠牲になったが、新規ユーザー獲得という長期的な投資を行った可能性がある。
- C. 売上が増加したので、営業利益の減少は問題ない。

---

### 問題 4：費用の分類

あなたが所属するテック企業で、以下の費用が発生しました。「販管費」に含まれる可能性が最も高いものはどれですか？

- A. ユーザーへの請求処理を行う決済代行会社への手数料
- B. サービスのインフラとして利用している AWS の利用料
- C. 新機能開発チームのエンジニアの人件費

---

## 解答と解説 | Answers and Explanations

### 問題 1：正解 A

**計算：**

1.  **売上総利益** = 売上高 - 売上原価 = 5,000 万円 - 1,000 万円 = 4,000 万円
2.  **営業利益** = 売上総利益 - 販管費 = 4,000 万円 - 2,500 万円 = 1,500 万円

**解説：**

営業利益は、本業での儲けを示す利益です。計算式は `売上総利益 - 販管費` です。営業外収益は経常利益の計算に使うため、営業利益の計算には含めません。

---

### 問題 2：正解 A

**解説：**

- **A 社** は粗利率が 20% と低いです。これは、売上原価（仕入れコストなど）が高いビジネス、例えば物理的な商品を販売する **小売業** や製造業の特徴です。
- **B 社** は粗利率が 85% と非常に高いです。これは、売上原価が低いビジネス、例えばソフトウェアを提供する **SaaS 企業** や、ライセンス販売を行う企業の典型的な特徴です。SaaS の場合、一度ソフトウェアを開発すれば、追加の顧客に提供するコスト（サーバー代など）は比較的低く抑えられます。

---

### 問題 3：正解 B

**解説：**

広告宣伝費という販管費を増やした結果、売上は増えたものの、それを上回るコスト増により営業利益は減少しています。

- **A は不適切**：失敗と断定するのは早計です。獲得したユーザーの LTV（顧客生涯価値）が高ければ、将来的に大きなリターンを生む可能性があります。
- **B が最も適切**：広告宣'便費は、短期的な費用であると同時に、将来の売上を作るための **投資** という側面を持ちます。経営陣が短期的な利益を犠牲にしてでも、市場シェアの拡大やブランド認知度の向上を選んだ、という戦略的な意図が考えられます。
- **C は不適切**：売上が増えても、利益が減少しているのであれば、その投資効率は慎重に評価する必要があります。

---

### 問題 4：正解 C

**解説：**

費用の分類は企業の会計方針によりますが、一般的には以下のように考えられます。

- **A と B**：決済手数料やサーバー費用は、売上に直接的に紐づくコストとして **売上原価** に計上されることが多いです。サービスを提供するために不可欠な変動費だからです。
- **C**：エンジニアの人件費は、特定の売上に直接紐づけるのが難しく、企業全体の運営に関わる費用として **販管費** に含まれるのが一般的です。（一部を研究開発費として分けることもあります）。
